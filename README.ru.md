# Российский IPTV плейлист
Коллекция общедоступных российских IPTV каналов, собранных из разных источников в один плейлист.

## Использование
Скопируйте ссылку на плейлист и вставьте в любой IPTV плеер.

* Основной
  ```
  https://naggdd.github.io/iptv/ru.m3u
  ```
* Радиостанции
  ```
  https://naggdd.github.io/iptv/radio.m3u
  ```
* Мультфильмы
  ```
  https://naggdd.github.io/iptv/cartoons.m3u
  ```
